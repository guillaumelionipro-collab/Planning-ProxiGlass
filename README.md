# ProxiGlass Agenda (React + Vite + PWA)

- PWA installable (manifest + service worker)
- Agenda complet : Année / Jour (drag&drop) / Liste
- LocalStorage, export CSV et .ics

## Lancer
```
npm install
npm run dev
```

## Tester en prod + PWA
```
npm run build
npm run preview
```
